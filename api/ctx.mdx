---
title: Context API
description: Complete reference for the Fiber Ctx interface and methods.
---

The `Ctx` interface represents the HTTP request and response context. It provides methods for accessing request data, sending responses, and managing the request lifecycle.

## Type Definition

```go
type Handler = func(Ctx) error
```

## Core Methods

### App

Returns the App instance.

```go
func (c Ctx) App() *App
```

<ResponseField name="*App" type="*App">
  Returns the Fiber app reference.
</ResponseField>

```go Example
app.Get("/", func(c fiber.Ctx) error {
    config := c.App().Config()
    return c.JSON(config)
})
```

### Context

Returns a standard context.Context.

```go
func (c Ctx) Context() context.Context
```

<ResponseField name="context.Context" type="context.Context">
  Returns a context.Context instance safe for async operations.
</ResponseField>

```go Example
app.Get("/", func(c fiber.Ctx) error {
    ctx := c.Context()
    go doAsyncWork(ctx)
    return nil
})
```

### SetContext

Set a custom context.Context.

```go
func (c Ctx) SetContext(ctx context.Context)
```

<ParamField path="ctx" type="context.Context">
  The context to associate with the request.
</ParamField>

## Request Methods

### Params

Get route parameters.

```go
func (c Ctx) Params(key string, defaultValue ...string) string
```

<ParamField path="key" type="string">
  The parameter name.
</ParamField>

<ParamField path="defaultValue" type="string" optional>
  Default value if parameter doesn't exist.
</ParamField>

<ResponseField name="string" type="string">
  Returns the parameter value.
</ResponseField>

```go Example
app.Get("/user/:id", func(c fiber.Ctx) error {
    id := c.Params("id")
    return c.SendString("User: " + id)
})
```

### Bind

Get the Bind helper for request parsing.

```go
func (c Ctx) Bind() *Bind
```

<ResponseField name="*Bind" type="*Bind">
  Returns the Bind helper instance.
</ResponseField>

```go Example
type User struct {
    Name  string `json:"name"`
    Email string `json:"email"`
}

app.Post("/user", func(c fiber.Ctx) error {
    user := new(User)
    if err := c.Bind().Body(user); err != nil {
        return err
    }
    return c.JSON(user)
})
```

### Locals

Store or retrieve local values.

```go
func (c Ctx) Locals(key any, value ...any) any
```

<ParamField path="key" type="any">
  The key to store/retrieve.
</ParamField>

<ParamField path="value" type="any" optional>
  The value to store.
</ParamField>

<ResponseField name="any" type="any">
  Returns the stored value.
</ResponseField>

```go Example

    c.Locals("user", "john")
    return c.Next()
})

    user := c.Locals("user").(string)
    return c.SendString("Hello " + user)
})
```

## Control Flow

### Next

Execute the next handler in the stack.

```go
func (c Ctx) Next() error
```

<ResponseField name="error" type="error">

```go Example

    fmt.Println("Before")

    fmt.Println("After")
    return err
```
